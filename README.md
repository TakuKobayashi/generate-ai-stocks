# generate-ai-stocks

生成AIに作らせたそれぞれの各種開発プロジェクトをそれぞれストックしていっています

## プロジェクト

| プロジェクト | 説明 |
|------|------|
| [chat-app](./chat-app/) | partykitでチャットアプリをサンプルとして作成したプロジェクト、様々なプラットフォームで動くサンプルを作成 |
| [daily-report-cli](./daily-report-cli/) | エンジニア向け、日報自動作成ツール |
| [doc-scanner-unified](./doc-scanner-unified/) | 画像から文字におこしてくれるCLIツール、Webサービス |
| [email-auto-reply](./email-auto-reply/) | メールの返答を自動的に作成してくれるツールプロジェクト |
| [gather-app](./gather-app/) | [Gather](https://www.gather.town/) のようなバーチャルワークスペースサービス、PartyKitとLiveKitを使ったWebSocketとWebRTCを使ったサービスのサンプル |
| [github-leak-detector](./github-leak-detector/) | Github上にソースコードが流出していないか検出するツール |
| [nomikai](./nomikai/) | 急に今日飲みに行こうと人を誘う・募集・通知するアプリ・webサイト |
| [recstudio](./recstudio/) | ブラウザ上で画面録画を行い動画ファイルとして保存することができるWebサイト。バックエンド不使用。 |
| [stamp-rally](./stamp-rally/) | スタンプラリーの作成と共有をして遊ぶことができるサービス |
