# generate-ai-stocks

生成AIに作らせたそれぞれの各種開発プロジェクトをそれぞれストックしていっています

## プロジェクト

| プロジェクト | 説明 |
|------|------|
| [chat-app](./chat-app/) | partykitでチャットアプリをサンプルとして作成したプロジェクト、様々なプラットフォームでの動くサンプルを作成 |
| [email-auto-reply](./email-auto-reply/) | メールの返答を自動的に作成してくれるツールプロジェクト |
| [gather-app](./gather-app/) | [Gather](https://www.gather.town/) のようなバーチャルワークスペースサービス、PartyKitとLveKitを使ったWebSocketとWebRTCを使ったサービスのサンプル |
| [stamp-rally](./stamp-rally/) | スタンプラリーの作成と共有をして遊ぶことができるサービス |
